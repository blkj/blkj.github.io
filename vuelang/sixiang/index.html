# 一门语言的学习成本

几年前我就知道vue，但是那时候装上vue后我什么也不能干，现在也干不了什么。

人如果不是因为过于懒惰的天性和孤单的现实，我觉得没人会喜欢代码。但是生在了代码掌控的时代只能如此。

人如果不是想法过多而求不得的痛苦过重也不会选择和代码死缠烂磨。

人如果不是现实所迫，个个都是天才，我在02年就计算机全国二级证书拿到手。那意味着一个人具备了学习代码的基本能力。可惜，一个人从入门到入坑放弃说的最典型的就是我。

我学习语言的成本是无限大，一辈子。

# 做网站是被人看不起的

很多人问你干什么，我从来不说我做网站，事实上我更多的是去工地干活，学习代码或者计算机都是我的兴趣爱好。2015前后经过四年死缠烂魔我拿到了全国软考网络工程师证，到现在也没挂出去，刚考上这不国家严打挂靠。

我的专业是工商管理，所以我一次就经济师中级就过了。但就是这样，我没有正儿八经的工作。所有的工作除了贩卖毒品估计都干过。除了不是工作的工作，农民工能干的我都能干。但是我没干过网站。

想起抠图，想起打杂，想起一个网络项目的复杂琐碎，想起曾经的黑白颠倒，腰肌劳损后背酸疼，想起通透一门语言的冲锋，我确定我不是计算机天才。

# 做项目就得光明正大

说做项目，很多人就把你当成撸代码复制粘贴的，那个程序员不是撸过来的，谁没有撸过，撸撸就习惯了。

做项目一天的机会成本和专业高度决定了，你要用一门语言做个东西，还真的和包工头没有区别。所以你要光明正大。

但是这条路本身就是工匠之路，和当前的浮躁格格格不入。另外进入中年我们说放弃就放弃，没有什么可惜的。人生就这么把美的的东西毁灭给人看。
